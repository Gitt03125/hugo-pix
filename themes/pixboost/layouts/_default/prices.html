Example of billing calculation

Original image size

Imagine, you have an Original image which weights 1MB on your website.

Image processing

Pixboost downloads it, optimises / transforms it on CDN for 30 days.

Image compression

We compress it to 600KB not losing the quality.

Users saw it

Let’s pretend that 1024 users saw it.

So, the calculation would be:
600Kb * 1024 views = 600MB of billed bandwidth
