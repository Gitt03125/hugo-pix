Limitations

We are providing Out-of-the-box Cloud Service, therefore we try to make as fewer limitations as possible.
  • Maximum image file size - 10 MB
  • 10 API Keys limit*
  • 10 Image Sources (domains) limit*
*If you need more API Keys or Image Sources please contact us directly and we will add more for your account.

Example of billing calculation
